The one I'm proudest of. I designed, built, and shipped a full-stack Solana marketplace end to end — product, engineering, and operations, all of it mine.

JavaScript and Node.js on Firebase. Secure auth with Firebase Authentication and role-based access, a Firestore backend served through serverless Cloud Functions and a RESTful API layer handling every marketplace transaction, and on-chain SOL payments powering peer-to-peer trades.
